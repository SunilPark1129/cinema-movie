### Installation
1. Clone the repository
```
git clone https://github.com/SunilPark1129/cinema-movie.git
```
2. ```cd``` into the clone folder
```
cd cinema-movie
```
3. Install the required dependencies
```
install npm
```
4. To run the project
```
npm start
```
